## `logs/`

The `logs/` directory contains runtime logs generated while using ApplyWise.

### Contents:

- API request and response logs (job saving, updates, retrieval)
- Error logs from job parsing or external integrations
- AI analysis logs (e.g., job matching or scoring outputs)
- Background task logs (reminders, scheduled checks)

<!-- BEGIN TREE -->
```text
├── app.log
└── README.md
```
<!-- END TREE -->


### Purpose:

Helps debug application behavior, monitor system health, and trace issues in job tracking or analysis workflows.
