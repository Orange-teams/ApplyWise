## `data/`

The `data/` directory stores structured data used by ApplyWise for processing, analysis, or initialization.

### Contents:

- Sample job postings used for testing parsing and analysis
- Seed data for users, applications, and job statuses
- Reference datasets (e.g., common job titles, industries, skill taxonomies)
- Cached or preprocessed job listings (if applicable)
